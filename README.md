# AstraPatch

AstraPatch is a comprehensive real-time space exploration monitoring system featuring advanced visual effects, optimized loading, and extensive tracking capabilities. Built with animated starfield canvas, floating planets, shooting stars, and cyberpunk aesthetics for an immersive space tracking experience.

## Enhanced Visual Features

- Animated canvas-based starfield with 500 parallax stars
- Three floating planets with realistic orbital animations
- Shooting stars that streak across the sky
- Pulsing nebulas in the background
- Professional loading screen with progress tracking
- Smooth card animations and transitions throughout
- Glitch effects on titles
- Hover animations with light sweeps
- Skeleton loaders for better perceived performance

## Performance Optimizations

- Progressive loading system with visual feedback
- Optimized canvas rendering at 60fps
- Debounced scroll events using requestAnimationFrame
- Lazy loading for images
- Error boundaries prevent site crashes
- Incremental data loading
- GPU-accelerated animations

## Core Features

### Rocket Launch Tracking
Real-time launch data with countdown timers, success probabilities, launch windows, and provider filtering for SpaceX, NASA, and other agencies.

### Live ISS Tracking
2D Earth visualization with real-time position updates every 5 seconds showing latitude, longitude, altitude, and velocity.

### Satellite Monitoring
Track Hubble, Starlink, Tiangong, and James Webb with live orbital data and positions.

### Space Station Crew
Current ISS crew with photos, agencies, and mission dates.

### Solar Activity
Solar flares, CMEs, and geomagnetic storms from NASA DONKI API.

### Celestial Events
Near-Earth asteroids, Mars weather, meteor showers, and moon phases.

### Astronomy Picture of the Day
Daily NASA APOD with high-resolution imagery and descriptions.

## Technical Stack

- Vanilla JavaScript (ES6+)
- Canvas API for starfield animation
- CSS Grid & Flexbox for responsive layout
- CSS Custom Properties for theming
- Multiple NASA and space APIs
- No external dependencies or build process


## Browser Support

Modern browsers with ES6+, Canvas API, CSS Grid support:
- Chrome (recommended)
- Firefox
- Safari
- Edge

Hardware acceleration recommended for smooth animations.

## API Sources
